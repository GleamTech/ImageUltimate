ImageUltimate offers 3 main classes:

  • ImageInfo class which allows you to read image information and metadata.

  • ImageTask class which allows you to process a source image in a chained manner and save the resulting image(s).

  • ImageWebTask class which can not be instantiated but which is passed as a parameter to your lambda expression so that you can chain image processing commands.
    ImageUltimate adds extensions methods to Page class for ASP.NET WebForms support: 

    • <%=this.ImageTag("Image.jpg", task => task.ResizeWidth(300))%>


    • <%=this.ImageUrl("Image.jpg", task => task.ResizeWidth(300))%>
